## 🎯 Scenario-Based Guide

### Scenario 1: "Saya langsung ingin buka dan test website"

1. Terminal: `npm run dev`
2. Browser: http://localhost:3000
3. Lihat website berjalan! ✨
4. Baca: `QUICK_START.md` untuk ubah content

**Total time: 2 minutes**

---

### Scenario 2: "Saya ingin customize sebelum deploy"

1. Baca: `QUICK_START.md` (5 min)
2. Edit `app/page.tsx` (10 min)
3. Add photos ke `public/images/` (5 min)
4. Test di mobile (5 min)
5. Baca: `DEPLOYMENT.md` (10 min)
6. Deploy ke Vercel (5 min)

**Total time: 40 minutes**

---

### Scenario 3: "Saya sudah familiar dengan Next.js & hanya ingin deploy"

1. Skim: `QUICK_START.md` (2 min - optional)
2. Edit content di `app/page.tsx` (5 min)
3. `npm run build` untuk verify (1 min)
4. Baca: Deploy section di `DEPLOYMENT.md` (5 min)
5. Push ke GitHub & deploy Vercel (5 min)

**Total time: 18 minutes**

---

### Scenario 4: "Saya tidak familiar, jelaskan semuanya!"

1. Baca: `README.md` (20 min)
2. Baca: `QUICK_START.md` (5 min)
3. Ikuti langkah-langkah di docs (30 min)
4. Baca: `DEPLOYMENT.md` (10 min)
5. Deploy! (10 min)

**Total time: ~75 minutes**
